# ThaiFex Sales Proposal Studio
React + Vite + TypeScript + @react-pdf/renderer

Project root is now this folder:

`/Users/Mod-Akrw/Dev/Web-App-Export-Doc-Gen`

## Quick Start
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Notes
- PDF export requires the CSP in `index.html` that allows `wasm-unsafe-eval`.
- Online client view is available at `/?view=client`.
- Share links intentionally omit embedded logo/signature images to keep the URL portable.
